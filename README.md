# أزرار شبكات التواصل الاجتماعي

يعرض هذا المشروع مجموعة من أزرار شبكات التواصل الاجتماعي مع تأثيرات التحويم باستخدام HTML و CSS. الأزرار تمثل المنصات الشهيرة: إنستغرام، تويتر، جيت هاب، وواتساب.

## المميزات

- تأثير التحويم: ترتفع الأزرار وتغير لونها عند المرور عليها بالماوس.
- تصميم عصري بأشكال وزوايا مميزة لكل زر.
- أيقونات SVG واضحة وقابلة للتكبير.
- ترتيب مرن باستخدام flexbox: صفين من الأزرار.

## طريقة الاستخدام

1. انسخ المشروع أو قم بتحميله.
2. افتح ملف `index.html` في المتصفح.
3. حرّك الماوس فوق الأزرار لمشاهدة تأثير التحويم.

## الألوان عند التحويم

- إنستغرام → أحمر (#fd1d1d)  
- تويتر → أزرق (#24a0ed)  
- جيت هاب → أصفر (#f7b733)  
- واتساب → أخضر (#4cd137)
